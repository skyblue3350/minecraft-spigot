# minecraft-forge
dockerized Minecraft Spigot Server

## quick start
### self build
```
$ git clone [this repo]
$ docker-compose up -d --build
```

### use docker hub image
```
$ curl -l -O https://raw.githubusercontent.com/skyblue3350/minecraft-spigot/[branch]/docker-compose.yml
$ docker-compose up -d
```
